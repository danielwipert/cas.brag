## Tips

- LinkedIn doesn't let you embed a link preview in a signature, but a clean URL works.
- For email signatures specifically, the one-line version travels best across mail clients.
- If you want a different angle per audience (recruiters / customers / engineers), swap the short description before the URL — the URL itself stays the same.
