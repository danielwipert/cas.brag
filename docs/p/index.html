For builders

Six principles for trustworthy agentic AI.

Design rules abstracted from BRAG, the retrieval‑augmented generation system with two LLM skeptics gating every answer. Take what’s useful — none of this requires BRAG.

  1. 01
    Type your evidenceEmbeddings give you “similar.” Type tags give you “relevant.”
  2. 02
    Numbers come through a separate channelThe LLM never writes a number it didn’t receive as a typed value.
  3. 03
    Add a refusal stepA clean “I don’t know” is the highest‑value behaviour an agent can have — and the hardest to ship.
  4. 04
    Run a skeptic against your own answerA second LLM tasked with refuting your first one catches what no eval set can.
  5. 05
    Make every decision loggedWhen someone asks “why did the system do that?” you should be able to show them, step by step.
  6. 06
    Degrade visiblyThe worst failure mode is a confident wrong answer; the second worst is a system that fails silently.
